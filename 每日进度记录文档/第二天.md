### 1. SpringBoot使用Validation校验参数

1. 引入依赖：

   ```xml
   <dependency>
       <groupId>org.springframework.boot</groupId>
       <artifactId>spring-boot-starter-validation</artifactId>
   </dependency>
   ```

2. 常见注解：

   | 注解         | 功能                                                         |
   | ------------ | ------------------------------------------------------------ |
   | @AssertFalse | 可以为null，如果不为null的话必须为false                      |
   | @AssertTrue  | 可以为null，如果不为null的话必须为true                       |
   | @DecimalMax  | 设置不能超过最大值                                           |
   | @DecimalMin  | 设置不能超过最小值                                           |
   | @Digits      | 设置必须是数字且数字整数的位数和小数的位数必须在指定范围内   |
   | @Future      | 日期必须在当前日期的未来                                     |
   | @Past        | 日期必须在当前日期的过去                                     |
   | @Max         | 最大不得超过此最大值                                         |
   | @Min         | 最大不得小于此最小值                                         |
   | @NotNull     | 不能为null，可以是空                                         |
   | @Null        | 必须为null                                                   |
   | @Pattern     | 必须满足指定的正则表达式                                     |
   | @Size        | 集合、数组、map等的size()值必须在指定范围内                  |
   | @Email       | 必须是email格式                                              |
   | @Length      | 长度必须在指定范围内                                         |
   | @NotBlank    | 字符串不能为null，字符串trim()后也不能等于""                 |
   | @NotEmpty    | 不能为null，集合、数组、map等size()不能为0；字符串trim()后可以等于"" |
   | @Range       | 值必须在指定范围内                                           |
   | @URL         | 必须是一个URL                                                |

3. 搭配全局异常处理来使用

   ```java
   // 参数校验异常
   @ExceptionHandler(MethodArgumentNotValidException.class)
   public Result<Void> handleValidException(MethodArgumentNotValidException e)
   {
       // 取第一条错误信息返回给前端
       FieldError fieldError = e.getBindingResult().getFieldErrors().get(0);
       String message = fieldError.getDefaultMessage();
       log.warn("参数校验失败：{}", message);
       return Result.fail(message, Result.CODE_BAD_REQUEST);
   }
   ```

### 2. 正则表达式校验邮箱和手机号

邮箱：`^[a-zA-Z0-9_-]+@[a-zA-Z0-9_-]+(\.[a-zA-Z0-9_-]+)+`

> `@`之前：大小写字母和数字和`_`和`-`_，不限长度
>
> `@`号之后域名：大小写字母和数字和`_`和`-`，不限长度
>
> `.`后：大小写字母和数字和`_`和`-`，可匹配多次，用于多级域名，如（@mail.google.com）

手机号：`^1[3-9]\d{9}$`

> 第一位：1
>
> 第二位：3-9
>
> 后面一共九位：0-9数字即可

### 3. lambda 查询

`lambdaQuery()` 或 `lambdaUpdate()` 中常用函数：

| 分类      | 方法                    | 说明                    |
| --------- | ----------------------- | ----------------------- |
| 基础筛选  | `.eq(col, val)`         | 等于 (=)                |
|           | `.ne(col, val)`         | 不等于 (<>)             |
|           | `.gt(col, val)`         | 大于 (>)                |
|           | `.ge(col, val)`         | 大于等于 (>=)           |
|           | `.lt(col, val)`         | 小于 (<)                |
|           | `.le(col, val)`         | 小于等于 (<=)           |
| 模糊查询  | `.like(col, val)`       | LIKE %值%               |
|           | `.likeLeft(col, val)`   | LIKE %值                |
|           | `.likeRight(col, val)`  | LIKE 值%                |
| 范围/集合 | `.between(col, v1, v2)` | BETWEEN v1 AND v2       |
|           | `.in(col, list)`        | IN (...)                |
| 排序/分页 | `.orderByAsc(col)`      | ORDER BY col ASC        |
|           | `.orderByDesc(col)`     | ORDER BY col DESC       |
| 逻辑拼接  | `.and(w -> ...)`        | AND（子条件）           |
|           | `.or()`                 | OR                      |
| 控制符    | `.isNull(col)`          | IS NULL                 |
|           | `.isNotNull(col)`       | IS NOT NULL             |
| 关键技巧  | `.eq(cond, col, val)`   | cond 为 true 才执行条件 |

### 4. Stream 的使用

`Stream<T>` 是 Java 8 引入的，可以理解为一个**数据流水线**，它本身不存数据，只负责对数据做一系列操作，`T` 是流里元素的类型

例子：

```java
// 合并两个 list 为 FileTreeNodeDto
List<FileTreeNodeDto> res = Stream.concat(
    folders.stream().map(folder ->
        {
            FileTreeNodeDto dto = new FileTreeNodeDto();
            dto.setId(folder.getId());
            dto.setName(folder.getName());
            dto.setParentId(folder.getParentId());
			dto.setType(FileTreeNodeTypeEnum.FOLDER);
            return dto;
        }),
    notes.stream().map(note ->
        {
            FileTreeNodeDto dto = new FileTreeNodeDto();
            dto.setId(note.getId());
            dto.setName(note.getTitle());
            dto.setParentId(note.getFolderId());
            dto.setType(FileTreeNodeTypeEnum.NOTE);
        })
).collect(Collectors.toList());
```

这里的例子中，先将`folders`和`notes`转化成 Stream 类型，然后使用`map()`，对每一个元素做这个操作，返回的是一个新的 Stream，然后使用`Stream.concat()`拼接成一个新的流，并且使用`.collect(Collectors.toList());`转化成所需的 List
